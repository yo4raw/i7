# 【非公式】アイドリッシュセブン 衣装データベース サイト仕様書

## 1. サイト概要

- **サイト名**: 【非公式】アイドリッシュセブン 衣装データベース / アイナナ攻略
- **URL**: https://i7.step-on-dream.net/
- **概要**: バンダイナムコオンライン提供のiOS/Android向けゲームアプリ「アイドリッシュセブン（通称：アイナナ）」の非公式情報データベースサイト。衣装（カード）のステータス・スキル情報の検索・比較、イベント特効情報、衣装ビジュアル一覧などを提供する。
- **対象レアリティ**: SSR・URの衣装データを中心に掲載
- **運営形態**: ユーザー参加型の編集可能データベース（認証なし・誰でも編集可能）
- **開発担当**: 悠（@you_at_i7）
- **関連サイト**: アイドリッシュセブン（アイナナ）攻略Wiki と情報の相互補完

| カラム名 | 説明 |
|---|---|
| ★ | お気に入りマーク（クリックで切り替え） |
| ID | 衣装ID |
| 衣装名/名前 | 衣装シリーズ名＋キャラクター名（カード詳細へのリンク） |
| レア | レアリティ（UR / SSR） |
| Shout | シャウト値（MAX値、属性一致時はカラーハイライト） |
| Beat | ビート値（MAX値、属性一致時はカラーハイライト） |
| Melody | メロディ値（MAX値、属性一致時はカラーハイライト） |

各言語版では、テーブルヘッダー・スキル名・UI テキストが翻訳されている。フィルター項目は名前（日本語のまま）と衣装名セレクトのみの簡略版。

### 3.5 スキル一覧ページ群

スキルタイプごとに、各衣装のLv1〜Lv5のスキル詳細を一覧表示する。

#### 3.5.1 スコアアップ衣装スキル一覧（`/scoreup.php`）

- **対象**: スコアアップスキルを持つ衣装
- **フィルター**: 名前（チェックボックス）、衣装名（セレクト＋フリーワード）
- **テーブルカラム**: ★、ID、衣装名/名前、レア、Shout、Beat、Melody、Lv1〜Lv5（各レベルのスキル詳細テキスト）
- **スキル詳細形式**: 「【Lv{n}】{発動条件}を{N}回毎に{N}％の確率でスコア{N}UP」
- 推測値には「（推測値）」の注記付き
- ページネーション: 100件/ページ

#### 3.5.2 縮小衣装スキル一覧（`/reduction.php`）

- **対象**: 縮小＆スコアアップスキルを持つ衣装
- **フィルター**: 同上
- **テーブルカラム**: 同上
- **スキル詳細形式**: 「【Lv{n}】{発動条件}を{N}回毎に{N}％の確率で{N}秒間縮小＆{N}倍UP」

#### 3.5.3 判定強化衣装スキル一覧（`/judgment.php`）

- **対象**: 判定強化スキル（BAD以上判定強化 / MISS以上判定強化）を持つ衣装
- **フィルター**: 同上
- **テーブルカラム**: 同上
- **スキル詳細形式**: 「【Lv{n}】{発動条件}{N}回毎に{N}％の確率で{N}秒間BAD以上をPerfectに」

### 3.6 スキル比較ページ群

スキルタイプごとに、楽曲別の効果予測値を含めた比較テーブルを提供する。各スキルレベルが個別の行として展開表示される。

#### 3.6.1 スコアアップ衣装スキル比較（`/scoreup_sort.php`）

- **テーブルカラム**: ★、ID、衣装名/名前、レア、Shout、Beat、Melody、スキル（レベル別詳細）、Treasure!加算値予測、幸せでいて加算値予測
- **特徴**: 楽曲ごとのスコア加算値予測を表示（「{N} UP」形式）
- **データ行数**: 約365行（各衣装×各スキルレベル）
- **フィルター**: 名前、衣装名

#### 3.6.2 縮小衣装スキル比較（`/reduction_sort.php`）

- **テーブルカラム**: ★、ID、衣装名/名前、レア、Shout、Beat、Melody、スキル（レベル別詳細）、Treasure!発動予測、幸せでいて発動予測
- **特徴**: 楽曲ごとの発動予測を表示（「{N}倍- {N}秒」形式）
- **データ行数**: 約527行

#### 3.6.3 判定強化衣装スキル比較（`/judgment_sort.php`）

- **テーブルカラム**: ★、ID、衣装名/名前、レア、Shout、Beat、Melody、スキル（レベル別詳細）、Treasure!発動秒数予測、幸せでいて発動秒数予測
- **特徴**: 楽曲ごとの発動秒数予測を表示（「{N} 秒発動」形式）
- **データ行数**: 約320行

## 9. データ編集フロー

サイトは認証なしで誰でも編集可能な参加型データベースである。

### 9.1 既存衣装の情報更新

1. カード詳細ページ下部の「情報更新画面へ」ボタンをクリック
2. 編集フォームで情報を入力（わかるものだけでOK）
3. 「データを更新する」ボタンで更新
4. 何回でも編集可能
5. ステータス値は桁区切りのコンマあり/なし両対応

### 9.2 画像登録

1. カード詳細ページ下部の「画像登録画面へ」ボタンをクリック
2. 画像ファイルを選択
3. 切り抜き範囲を4方向ボタンで微調整
4. 「衣装画像をアップロード」ボタンで登録

### 9.3 新規衣装シリーズ登録

1. 「データ管理」→「衣装新シリーズ登録」フォームへ
2. レアリティごとの衣装名とシリーズ名を入力
3. 登場グループをチェックボックスで選択（一部のみの場合は「その他」で個別指定）
4. 衣装実装日を入力
5. 「衣装シリーズを登録する」で、指定キャラクター分の個別データ枠を作成

### 9.4 新規イベント登録

1. 「イベント特効」→「イベントを新しく登録」ボタン
2. イベント名、日程を入力し、特効衣装の効果を選択
3. 準々特効（銅特効）がある場合はメンバー名を入力
4. 「イベント新規登録」で登録

### 9.5 イベント特効衣装登録

1. イベント特効ページから対象イベントを選択
2. 「特効衣装を登録」「準特効衣装を登録」をクリック
3. 衣装名のドロップダウンで絞り込み、チェックをつけて登録

### 9.6 データ除外

1. カード詳細ページ下部の「この情報を一覧から削除」ボタン
2. 確認アラートでOKを選択
