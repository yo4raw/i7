
# SongsのID2のMonsterGenerationでデッキがセンターにCardのID3414のみだった場合のスコア計算方法
- スキルのタイプがBAD以上をPerfectに変更なのでスコアに寄与しない
-- そのためMCシミュレーションも変化なし

## CardのID3414のみの属性値
- 未特訓の場合
-- Shout 3,898
-- Beat 5,891
-- Melody 4,611
- 特訓済みの場合
-- Shout 3,898
-- Beat 7,691
-- Melody 4,611
-　固定ブローチを考慮した場合

## デッキ全体の属性値
- 固定ブローチを考慮
-- CardがID3414のみなので条件を満たす beatに+4500
- センターボーナス
- URなのでBeatに7,691+4500の10%である1219を加算
