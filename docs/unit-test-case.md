##　スコア計算の概要
Notes 2.0の「S白」は「Shout属性値」* 1 * 2.5%
Notes 2.0の「S色」は「Shout属性値」* 1 * 3%

## 1ノートあたりのスコア算出式

実装 `calcNoteScore` は 2 段階の floor を適用する。

```
rawNote  = floor(属性値 × NOTE_RATE[white|color])
noteScore = floor(rawNote × LIGHT_MULTIPLIER[group])
```

- `NOTE_RATE.white = 0.025`、`NOTE_RATE.color = 0.030`
- 各ノート種別の 1 ノートあたりスコア(`Shout=3,898 / Beat=13,410 / Melody=4,611` 時)

| 種別 | 属性値 | rawNote |
|------|------:|------:|
| S白 | 3,898 | floor(3,898 × 0.025) = **97** |
| S色 | 3,898 | floor(3,898 × 0.030) = **116** |
| B白 | 13,410 | floor(13,410 × 0.025) = **335** |
| B色 | 13,410 | floor(13,410 × 0.030) = **402** |
| M白 | 4,611 | floor(4,611 × 0.025) = **115** |
| M色 | 4,611 | floor(4,611 × 0.030) = **138** |
