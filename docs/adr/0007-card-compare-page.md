# 0007: 衣装比較ページの新設とスキル種別ごとの比較軸

- 日付: 2026-06-11
- ステータス: 承認（2026-06-11 ユーザー承認、モックアップ確認済み）

## 背景

衣装単体の「強さ」は属性値・スキル・曲に依存し、どちらの衣装がスコアを伸ばすか比較しにくい。特に判定縮小スキルはデッキ全体（2枚以上で発動・並び順）に依存するため、単一数値での「単体強さ」はそもそも定義できない。世間の検証画像（曲固定の衣装別スコア棒グラフ）も縮小の扱いが不明で検証不能だった。

## 決定

前提条件を明示的に固定した専用ページ `/card-compare/` を新設し、**スキル種別ごとに異なる比較軸**で全衣装をランキング表示する:

- **スコアアップ系**（Perfect / コンボ / タイマー + 判定補正系・スキルなし）: 「属性値由来スコア + スキル期待値」の積み上げ棒グラフで降順比較。Perfect 前提なら全て同じ土俵で期待値比較できるため
- **判定縮小系**: 期待スコア化せず、実際の評価基準に合わせた多段ソート（効果秒数 → 必要コンボ数 → 確率 → 属性値）。同率は1列に縦積みして視覚化
- 固定する前提: UR 限定 / Perfect 前提 / センタースキル無視 / 固有ブローチ装備込み / 曲選択式（初期 DIAMOND FUSION）/ デフォルト所持衣装のみ / 特効はトグル（デフォルトオフ）

ランキングから最大4枚をピックアップして数値内訳・スペックを横並びにする詳細比較パネルを併設する（ハイブリッド構成）。

## 検討した代替案

- **縮小スキルの単一指標化**（発動率 × 秒数 × 縮小段階の合成値）— 合成値の意味の説明が必要になり、実際の評価手順（秒数優先の辞書式比較）と乖離するため不採用
- **スコア計算ページへのモード追加** — 既存ページが編成 UI で複雑であり、比較は「編成前の検討」という別ジョブのため不採用
- **スコアアップと縮小の統合ランキング** — 役割が違い横並び比較する場面がないため不採用
- **センタースキルの「センターに置いた場合」表示モード** — 不要との判断で完全無視に確定
